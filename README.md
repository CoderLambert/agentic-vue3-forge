# agentic-vue-forge

A pragmatic Vue3 foundation for human-AI agentic engineering.

This project is a Vue3 engineering base template designed for long-term collaboration between humans and AI assistants. It emphasizes modular architecture, stable search paths, explicit responsibility boundaries, composable UI patterns, testability, and extensible skill-driven workflows.

Core goals:

- predictable module structure
- clear API / Query / Mutation / Store boundaries
- stable naming conventions
- scalable UI composition
- review-friendly AI output
- plugin-style expansion for Tailwind, shadcn, Vitest, Storybook, and more
